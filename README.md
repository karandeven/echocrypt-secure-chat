EchoCrypt is a secure real-time chat application developed as a final year engineering project.
The system ensures privacy by encrypting messages on the client side before transmission,
so that even the server cannot read the message content.

### Key Features
- Real-time one-to-one messaging
- Client-side end-to-end encryption
- Secure key exchange mechanism
- Privacy-first architecture
- Simple and clean user interface

### Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Real-time Communication: Socket.IO
- Encryption: AES & RSA (Web Crypto API)
